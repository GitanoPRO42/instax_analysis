# (eng)  instax_sales_analysis
Project to analyze my Instax photo sales while learning Python, SQL, and Git. 

Structure      

    data/: raw and processed data

    scripts/: Python code

    notebooks/: Jupyter experiments

    outputs/: results and charts


Objectives

    Weeks 1-2: loading and basic analysis with Python

    Weeks 3-4: SQL queries on the data

    Weeks 5-6: automation and visualization
